Smart Water Coaster

[October 2022]

Drink coaster that tracks water consumption throughout the day using a microcontroller and will alert the user hourly via SMS and LEDs if hydration levels not met

  • Skills: Microcontrollers, C, Twilio API, Soldering, OnShape,

Features

Microcontroller

A Particle Photon microcontroller was utilized because of its Wi-Fi capabilities, and it was soldered to a force sensing resistor (FSR). The microcontroller was also soldered to wires that attached to 6 LEDs. The Photon could collect data and wirelessly send it to another device via Bluetooth

CAD

The CAD base was an iterative process using SolidWorks and calipers for measurement. The base contained the electronic hardware of the project, and it consisted of 3 parts: a top, a bottom, and a middle piece to consistently distribute the bottle's weight across our sensor.

Sensor Calibration

To calibrate the sensor and find a relationship between voltage readings from the sensor and volume of water, a linear regression was used. This created an optimal relationship to most accurately convert the sensor readings to water consumption.

Code and Twilio API

The code for the Photon microcontroller and to integrate the whole project was written in C++. The Twilio API was used in conjunction with a webhook in order to send an hourly text message to a specified phone number. The code is linked in the GitHub below. Readings were constantly taken by the sensor, and if the voltage readings dropped by a certain threshold every hour, the device registered enough water had been consumed. Otherwise the LEDs would light up and a SMS would be sent.
