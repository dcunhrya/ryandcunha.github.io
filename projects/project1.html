[January 2022 - Present]

Computer Vision based device that uses Mediapipe machine learning framework model and Tensorflow GPU inference to monitor a squat in real-time while providing visual cues, form improvement feedback, and other analysis.

  • Skills: Python, Microprocessors, TensorFlow

Project Goals

    • Monitor squat in real time and determine sufficient squat depth
    • Provide audio and visual cues
    • Analyze where squat potentially went wrong


Future Plans

    • Build and integrate hardware for device
    • Create app to analyze squats and interface with device
    • Allow users to enable features and customize feedback based on squat

Hardware and Dependencies

    • NVIDIA Jetson Nano
    • Jetson Camera Module
    • Tensorflow
    • Mediapipe API
    • OpenCV

Features

Software

All code was written in Python and run on NVIDIA Jetson Nano. A logger was set up to assist in debugging when running code on the microprocessor. Squat state transition logic was used to determine stage of squat based on the angles between limbs.

Machine Learning

In order to determine squat depth and phases, limbs are derived from certain pose landmarks estimated using Mediapipe Machine Learning frameworks. This pose inference is spedup by Tensorflow working with Mediapipe to perform CPU and GPU inference.
