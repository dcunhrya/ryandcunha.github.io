[August 2024 - December 2024]

Conditional variational autoencoder to generate images from long text using CLIP on COCO and short text on Fashion-MNIST. Final project from Duke CS675: Deep Learning

  • Skills: PyTorch, Deep Learning

Final Report


Key Results

Architecture Design

Two main conditional VAE architectures were tested. The first architecture involved a fully connected design mainly used for short text image generation and reconstruction on Fashion-MNIST. Other experiments used a convolutional VAE architecture. Full details can be found in the GitHub repo. This architecture was used to train with the COCO dataset along with a CLIP feature extractor before being passed to the conditional VAE. For all conditioning, the input was inserted into the bottleneck layer. Models were built to handle images of size 64x64, 128x128, and 256x256.

Experiment Design

The first experiment involved short-text model architecture investigation between fully connected and a convolutional CVAE, where the convolutional CVAE was found to have a lower MSE and higher Structural Similarity Index Measure (SSIM) and Frechet Inception Distance (FID). The second experiment tested long-text image generation and reconstruction on images of size 64x64, 128x128, and 256x256. The third experiment tested concatenating versus adding the conditional input in the CVAE from the COCO annotation using the features extracted by CLIP.

Short Text to Image

Image reconstruction and generation from Fashion-MNIST dataset with images of size 28x28. All figures and results are shown in the final report document in Section 4. Tested with concatenating and additive conditional inputs.

Long Text to Image

Image reconstruction and generation from COCO dataset using CLIP feature extractor. Built for images of size 64x64, 128x128, and 256x256. All figures and results are shown in the final report document in Section 4 and 5. Tested with concatenating and additive conditional inputs.
